DankNet Menu

An Open-Source C# GTA V menu, based on ScriptHookVDotNet.

See pictures of it here (some pictures can be old, and those options be changed, moved or improved)

Also I don't provide virus scans, do them on virustotal etc yourself.

How to install:

0) Understand that you'll use it at your own risk.

Skip to 2 if you have ScriptHookV, skip to 3 if you have both SHV.Net and SHV.

OR just watch the video: http://youtu.be/JjGEEMcizwc

1) Download and install ScriptHookV (or if you keep not listening to me and want to use it online, go here)

2-1) Download ScriptHookVDotNet's latest release (available here) / download current version's source code and compile (both will work, you pick.)

2-2) Install it by copying the .asi and .dll file to GTA V folder.

3) Download this project as zip and copy the .cs and .txt files (not license and readme.md) to the /scripts/ folder under your GTAV folder. (if it doesn't exist, create it)

Done. It opens with F6 key.
